# Data Structures and Algorithms

##### Patterns
```
├── Binary Tree/Binary Search Tree(BST)
│   ├── Traversal
│       ├── Pre-Order(ROOT-LEFT-RIGHT)
│       ├── In-Order(LEFT-ROOT-RIGHT)
            ├── Recursive, Stack, Iterative, Morris
                ├── Kth smallest element(BST)
                ├── Kth largest element(BST)(Reverse In-order(RIGHT-ROOT-LEFT))
                ├── Flatten Binary tree(Reverse In-order(RIGHT-ROOT-LEFT))

```
